# ⚔️ LingoSaga **英会話 × RPG ─ あなたの英語修行を、一人の冒険者の成長物語（サーガ）に。** ChatGPTのAdvanced Voice Modeで英会話し、その結果を記録してレベルアップしていく個人用トレーニングゲームです。データはすべて**あなた自身のGoogleスプレッドシート**に保存されます。開発者を含む第三者があなたのデータにアクセスすることはありません。 ## 🎮 ゲームの仕組み **二層構造：** | | ⭐ 実力レベル（Lv.1〜999） | 🎖️ 冒険者ランク（G〜S） | |---|---|---| | 意味 | 今の本当の腕前（測定値） | 積み上げた修行の証 | | 変動 | 調子により上下する | 絶対に下がらない | - 会話するたびに9つのパラメーター（発音・語彙・文法・流暢さ・聞き取り・表現力・自然さ・積極性・場の雰囲気）で採点 - セッションごとに「今日の戦い方」を職業判定（⚔️戦士・🧙魔法使い・🙏僧侶・🃏遊び人・👑賢者 など全9職） - EXPを貯めてランクアップ。最初の10回はチュートリアル期間 - 最上位ジョブへの転職条件は**現実の実績のみ**（試験合格・海外一人旅・登壇など）。ゲームは修行の場、本番は現実。 詳細は [docs/design.md](docs/design.md) を参照。 ## 🚀 セットアップ（約10分） ### 1. スプレッドシートとGASを用意 1. 新規Googleスプレッドシートを作成 2. メニュー「拡張機能 → Apps Script」を開く 3. 初期コードを全て消し、このリポジトリの `code.gs` を貼り付けて保存 4. 関数 `setup` を選んで ▶ 実行（初回は承認ダイアログを許可） 5. 「デプロイ → 新しいデプロイ → ウェブアプリ」 - アクセスできるユーザー：**全員** - デプロイしてURLをコピー ### 2. アプリにURLを設定 1. [ダッシュボード](https://kosa-e-mon.github.io/lingo-saga/) を開く 2. 設定欄にGASのURLを貼って保存（relay.htmlと共通） ### 3. GPTsを作成 1. ChatGPTで「GPTを作成」 2. Instructionsに `docs/gpts_prompt.md` の内容を貼り付け 3. Advanced Voice Modeで会話開始！ ### 4. プレイの流れ ``` GPTsと英語で会話（5〜15分） 　→ 終了時にJSONが出力される 　→ セッション記録画面（relay.html）に貼り付けて送信 　→ EXP獲得！ダッシュボードで成長を確認 ``` ## ⚙️ 運用メモ - GASのコードを更新したときは「新しいデプロイ」ではなく既存デプロイの**バージョン更新**を選ぶこと（URLが変わらない） - 最初からやり直したいときは、Apps Script画面から `resetAllData` を実行 ## 📜 クレジット 企画・開発：Masato Kosaka（[@Kosa-E-Mon](https://github.com/Kosa-E-Mon)） 開発協力：Claude (Anthropic) ## License MIT
